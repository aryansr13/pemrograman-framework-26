6. Refleksi & Diskusi 

1. Mengapa <img> biasa tidak optimal?
:<img> biasa tidak optimal karena tidak ada optimasi otomatis (lazy loading, resize, format), jadi loading lebih berat.

2. Apa perbedaan font CDN dan next/font? 
:Font CDN ambil dari internet (butuh request), sedangkan next/font di-optimasi lokal (lebih cepat, tanpa request tambahan).

3. Mengapa script bisa membuat website lambat?
:Script bisa bikin lambat karena blocking render dan menambah beban loading JS.

4. Kapan harus menggunakan dynamic import? 
:Dynamic import dipakai saat komponen berat atau tidak selalu dipakai (biar load saat dibutuhkan saja).

5. Apa dampak bundle size terhadap UX? 
:Bundle size besar → loading lama → UX jelek (lemot, delay).
